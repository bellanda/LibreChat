## #4: Carregamento de `file_search`

**Arquivo**: `api/app/clients/tools/util/fileSearch.js` (linhas 45-65)

```javascript
const primeFiles = async (options) => {
  const { tool_resources, req, agentId } = options;
  const file_ids = tool_resources?.[EToolResources.file_search]?.file_ids ?? [];
  const agentResourceIds = new Set(file_ids);
  const resourceFiles = tool_resources?.[EToolResources.file_search]?.files ?? [];

  // Get all files first
  const allFiles = (await getFiles({ file_id: { $in: file_ids } }, null, { text: 0 })) ?? [];

  // Filter by access if user and agent are provided
  let dbFiles;
  if (req?.user?.id && agentId) {
    dbFiles = await filterFilesByAgentAccess({
      files: allFiles,
      userId: req.user.id,
      role: req.user.role,
      agentId,
    });
  } else {
    dbFiles = allFiles;
  }

  dbFiles = dbFiles.concat(resourceFiles);

  // ← AQUI o prompt é gerado dinamicamente:
  let toolContext = `- Note: Semantic search is available through the ${Tools.file_search} tool but no files are currently loaded. Request the user to upload documents to search through.`;

  const files = [];
  for (let i = 0; i < dbFiles.length; i++) {
    const file = dbFiles[i];
    if (!file) {
      continue;
    }
    if (i === 0) {
      toolContext = `- Note: Use the ${Tools.file_search} tool to find relevant information within:`;
    }
    toolContext += `\n\t- ${file.filename}${
      agentResourceIds.has(file.file_id) ? '' : ' (just attached by user)'
    }`;
    files.push({
      file_id: file.file_id,
      filename: file.filename,
    });
  }

  return { files, toolContext };
};
```

**O que acontece**:
- Se não há arquivos: "no files are currently loaded"
- Se há arquivos: lista cada um com nome do arquivo
- Retorna `toolContext` que será salvo em `toolContextMap['file_search']`

## 🎯 EXEMPLO REAL: User Pede "Execute Python"

```
1. User: "Execute este código Python: print('hello')"

2. LibreChat vê: tool = 'execute_code' na agent.tools

3. handleTools.js carrega:
   - Cria CodeExecutionTool
   - Salva toolContextMap['execute_code'] = "Execute Python..."

4. run.ts monta:
   systemContent = "# `execute_code`: Execute Python...\nYou are a helpful AI..."

5. LLM recebe:
   System: "...`execute_code`: Execute Python...\n..."
   Tools: [{ name: 'execute_code', description: '...', schema: {...} }]
   User: "Execute este código Python: print('hello')"

6. LLM pensa:
   "User quer executar Python. 
    Tenho tool 'execute_code' que faz exatamente isso.
    Vou chamar com language='python' e code='print(\"hello\")'"

7. LLM retorna:
   { tool_name: 'execute_code', arguments: { code: 'print("hello")', language: 'python' } }

8. LangGraph executa:
   POST http://sandbox:3082/exec
   Headers: { 'X-API-Key': '...' }
   Body: { code: 'print("hello")', language: 'python' }

9. Sandbox responde:
   { stdout: "hello\n", stderr: "", exitCode: 0 }

10. LLM vê resultado e responde:
    "Executado com sucesso! Output: hello"
```
