## 📊 FLUXO: Arquivo → Arquivo → Arquivo

```
1. api/models/Agent.js
   └─ Define: agent.tools = ['execute_code', 'web_search', ...]

         ↓↓↓

2. api/server/controllers/agents/client.js
   └─ Chama: loadTools({ tools: agent.tools, ... })

         ↓↓↓

3. api/app/clients/tools/util/handleTools.js
   └─ Para cada tool:
      ├─ Se execute_code: 
      │  ├─ Chama primeCodeFiles()
      │  └─ Salva toolContextMap['execute_code']
      ├─ Se web_search:
      │  └─ Salva toolContextMap['web_search'] = "# `web_search`: ..."
      └─ Se file_search:
         ├─ Chama primeFiles()
         └─ Salva toolContextMap['file_search']

   Retorna: { loadedTools, toolContextMap }

         ↓↓↓

4. packages/api/src/agents/run.ts
   └─ createRun() recebe tools + toolContextMap
      ├─ systemMessage = toolContextMap.values.join('\n')
      ├─ systemContent = systemMessage + instructions
      └─ Passa para createAgentInput()

         ↓↓↓

5. @librechat/agents (lib externa)
   └─ Run.create() recebe:
      ├─ tools: [tool1, tool2, ...] com schemas
      ├─ instructions: systemContent (com descriptions)
      └─ Cria LangGraph

         ↓↓↓

6. LLM
   └─ Recebe:
      ├─ System prompt (com descriptions)
      ├─ Tool schemas
      └─ User message
      
      ↓ LLM decide qual tool usar

7. LangGraph invoca tool
   └─ Executa (e.g., POST sandbox:3082/exec)
      ↓ Retorna resultado
      ↓ LLM continua raciocínio
      └─ Responde ao usuário
```

## 📦 DEPENDÊNCIAS EXTERNAS

As seguintes estão **FORA** do repo mas controlam o comportamento:

| Pacote | Função | O que controla |
|--------|--------|----------------|
| `@librechat/agents` | `createCodeExecutionTool()` | Descrição do execute_code |
| `@librechat/agents` | `createSearchTool()` | Descrição do web_search |
| `@librechat/agents` | `Run.create()` | Como LLM é invocado com tools |
| `langchain` | `tool()` | Como tools são definidos em geral |

Se você quiser mudar o comportamento dessas, precisa:
1. Fork/mod o pacote npm
2. Ou copiar código para `api/`
3. Ou usar hooks/patches
